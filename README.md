Проект: Обучение с учителем — прогноз снижения покупательской активности
Цель проекта:
Предсказать, снизится ли покупательская активность клиента в ближайшие 3 месяца, и определить рекомендации для удержания клиентов с учётом их прибыльности.

Задачи:
- Построить модель классификации для прогноза активности.
- Провести исследовательский анализ данных.
- Очистить и подготовить данные.
- Объединить таблицы из разных источников.
- Обучить несколько моделей и сравнить их по метрикам.
- Определить наиболее важные признаки.
- Провести сегментацию покупателей.
- Сформировать предложения по удержанию клиентов.

Этапы работы:

- Обзор данных и проверка их качества.
- Предобработка: работа с пропусками, корректировка типов, преобразование признаков.
- Исследовательский анализ данных.
- Объединение таблиц в единый датасет.
- Обучение моделей с использованием пайплайнов.
- Оценка качества и выбор лучшей модели.
- Анализ важности признаков.
- Сегментация покупателей по прибыльности и активности.

В процессе решения был использован перебор через GridSearch следующих моделей:

- Logistic Regression
- SVC
- KNN
- Decision Tree

Лучший результат показала модель SVC:
SVC(C=1, probability=True, random_state=42)
Лучший ROC-AUC: 0.90
